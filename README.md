# 🌱 Planteki PWA

Planteki is a Progressive Web App (PWA) built with **React + Expo** that helps users explore plants, manage their profiles, and access plant‑related resources. It includes Firebase authentication and a clean modular UI.

---

## 🚀 Features
- **Authentication**: Sign up, sign in, and password reset via Firebase.
- **Splash & Login Screens**: Simple entry flow for new and returning users.
- **Dashboard**: Central hub for navigation.
- **Plant View & Plant Wiki**: Browse plant details and wiki pages.
- **Profile Management**: Editable username and bio, with logout support.
- **Cross‑platform**: Runs on web, Android, and iOS via Expo.
